# Step Definitions - Popups

[← Retour à l'index](../../STEP_DEFINITIONS.md)

Documentation des step definitions pour les popups/modales.

## Actions

| Step Definition | Arguments | Description |
|----------------|-----------|-------------|
| `Quand je ferme la popup` | Aucun | Ferme la popup |
| `Quand je ferme la popup deux fois` | Aucun | Ferme 2 popups imbriquées |
| `Quand je sauvegarde la popup` | Aucun | Clique sur sauvegarder |
| `Quand je valide la popup` | Aucun | Clique sur valider |
| `Quand je sauvegarde et effectue l'action de la popup` | Aucun | Sauvegarde et action |
| `Quand j'annule la popup` | Aucun | Annule et ferme |
| `Quand j'ajoute depuis la popup` | Aucun | Clique sur ajouter |

## Vérifications

| Step Definition | Arguments | Description |
|----------------|-----------|-------------|
| `Alors la popup devrait être ouverte avec le titre {string}` | Titre | Vérifie ouverture et titre |
| `Alors la popup devrait être fermée` | Aucun | Vérifie fermeture |
| `Alors le titre de la popup devrait être {string}` | Titre | Vérifie le titre |
| `Quand je clique sur le titre de la popup` | Aucun | Clique sur le titre |

## État des Boutons

| Step Definition | Arguments | Description |
|----------------|-----------|-------------|
| `Alors le bouton de validation devrait avoir le nom {string}` | Nom | Vérifie le texte du bouton |
| `Alors le bouton sauvegarder et action devrait avoir le nom {string}` | Nom | Vérifie le texte |
| `Alors le bouton de validation devrait être désactivé` | Aucun | Vérifie désactivation |
| `Alors le bouton de validation devrait être activé` | Aucun | Vérifie activation |
